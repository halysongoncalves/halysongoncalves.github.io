Code

Java for Android, Objective-C for iOS. Native applications perform better and are much more pleasing to the user. The center of development is always the user experience.

Design

The most important part of an application's design! Always thinking first in the user, the apps always have native design and well thought out for each application.

Platforms

No use to create application for Android and copy for all platforms. Each has a design and different type of user. Keeping most of the logic in WebService and deploying the application natively is superior quality assurance.
